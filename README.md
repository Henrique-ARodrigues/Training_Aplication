## Compilar e executar
Para compilar e executar o sistema, é necessário estar na pasta /app.
### Compilar
```
make
```
### Executar
```
make run
```
### Limpar
```
make clean
```
## Testes
Criamos um conjunto de testes manuais para conferir se as funcionalidades estavam corretas sem ser necessário executar sempre o menu
### Execução
```
make tests
make run_tests
```
